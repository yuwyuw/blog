git

一、本地项目放入github（https）

1. 登录github，创建一个仓库，复制仓库地址,打开命令行终端，进入项目所在的本地目录，将目录初始化为一个 Git 项目
2. 将代码放入git 暂存区
3. 讲代码提交到工作区
4. 回到命令行终端界面，将本地仓库关联到远程仓库
5. 提交代码到 GitHub 仓库
